\textbf{写像としての正則関数}

　

関数の概略をつかむにはそのグラフを描いてみるのがよい手がかりになる．
あいにく複素関数は変数$z$が2次元，関数のとる値$w=f(z)$も2次元
の空間を動くので，3次元の空間にグラフを描いて眺めるわけにはいかないが．
色々な曲線がどう写されるかを追跡してみると大体の様子がわかる．
図\ref{figBx1}は写像$w=z+1/z$の様子を示す．

\begin{figure}[hbtp]
	\centering
	\includegraphics[bb=0 0 800 200,clip,width=12cm,height=6cm]{figBx1.bmp}
	\caption{写像$w=z + 1/z$．$z$平面の単位円は$w$平面の線分$[-2,2]$に2重に移る
半径$k,1/k$の円は楕円に，原点を通る直線群は双曲線群に写される．
}
	\label{figBx1}
\end{figure}

図から，互いに直交する曲線の族(同心円の族および原点を通る直線の族）
が，再び直交する曲線の族(楕円の族とと双曲線の族)に写されること
が見て取れるだろう．一般に点$z_0$を通る2曲線は$w=f(z)$によって
$w_0=f(z_0)$を通るⅱ曲線にうつされるが，
$f'(z_0) \ne 0$ならば，それらの曲線のなす角度は不変に保たれる．
これは正則関数による写像の基本性質で，\textbf{等角性}とよばれる．

正則関数による写像によって，領域の形はどう変わりうるのだろうか．
一般に，2つの領域${\cal D}_1,{\cal D}_2$の間に，全単射$f:{\cal D}_1 \rightarrow {\cal D}_2$
であって$f$および逆写像がともに連続なものがあるとき，
${\cal D}_1,{\cal D}_2$は\textbf{位相同型}であるという．上でさらに
$f(z)$が正則ならば，逆写像も自動的に正則になることが知られている．
この場合2つの領域は\textbf{領域的に同型}であるという．
解析的に同型ならばもちろん位相同型であるが，後者ははるかに
後者ははるかに寛大な概念である．領域$\cal D$内の任意の
閉曲線が連続的に1点にまで変形できるとき$\cal D$は
\textbf{単連結}であるという($\cal D$に「穴」のないことと
思って良い)，単連結という性質は位相同型で，そして，
解析的同型で不変である．

\例{}
領域$\{z \: | \: r < |z| < 1 \: \}$は単連結でないから
単位円板$\mathbb{D} = \{ z \: | \: |z| <1 \: \}$と
位相で同型ではない．

これに対し，複素平面$\C$と$\mathbb{D}$は位相同型である（同型写像を
与える写像として，例えば$w = z / (1+|z|)$をとればよい．）
ところが，これらは解析的には同型ではない．実際，
$f : \C \rightarrow \mathbb{D}$なる正則関数は全平面で有界だから，
リウビルの定理によって定数しかあり得ない．

\□

実は次の事実が成り立つ．

\textbf{リーマンの写像定理　}
領域${\cal D}$が単連結で全平面$\C$とはことなるならば，
${\cal D}$は単位円板$\mathbb{D}$と解析的に同型である．
\□
