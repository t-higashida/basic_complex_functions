\subsection{一意接続の原理}

解析関数の各点での展開は，一般に定義領域の一部でしか収束しない．
ところが，関数が解析的である限り，1点での展開から遠くの点の値も
自然に決まってしまうのである．すなわち次の事実が成り立つ．

\定理{2.38}（一致の定理）
$f(z),g(z)$は領域$D$上で解析的であるとし，ある点$a	\in	D$に収束する点列
$z_n	\rightarrow	a	\:	(z_n	\ne	a,	z_n	\in	D)$の上で値が一致しているとする：
\[
	f(z_n)	=	g(z_n)	\quad	(	n	=	1,2,	\cdots).
\]
このとき，$D$のすべての点で$f(z)	=	g(z)$が成り立つ．特別の場合として，
\begin{list}{}{}
	\item	（ⅰ）ある点の近傍で，
	\item	（ⅱ）ある線分（例えば実軸の区間）の上で，
\end{list}
恒等的に成り立つならば，$D$全体で成り立つ．

\証明
$f(z)-g(z)$を改めて$f(z)	-g(z)$を改めて$f(z)$とすれば，$g(z)$が恒等的に$0$の場合を
示せばよい．

まず，$z=a$の近くで$f(z)	\equiv	0$が成り立つことを示そう．$f(z)$は解析的であるから，
$a$を中心としたべき級数展開をもつ：
\begin{equation}
	f(z)	=	a_0	+	a_1	(z-a)	+	a_2	(z-a)^2	+	\cdots
	\label{eq2x34}
\end{equation}
ここで，仮定により$f(z_n)=0$だから
\begin{equation*}
	a_0	=	f(a)	=	\lim_{n\rightarrow	\infty}	f(z_n)	=	0.
	\tag{\text{☆}}
	\label{eq}
\end{equation*}
（解析関数は連続）
このとき$f_1	(z)	=	f(z)/(z	-	a)	=	a_1	+	a_2	(z-a)	+	\cdots	$は
(\ref{eq2x34})と同じ収束半径を持つ（命題2.4）．
しかも$z_n	\ne	a$だから(分母は$0$にならず）$f_1(z_n)	=0$．
ゆえに再び$n	\rightarrow	\infty$とすれば（$z_n	\rightarrow	a$）
$a_1	=	f_1(z)=0$を得る．これを繰り返せば，結局$a_n	=	0$がすべての
$n$について導かれる．

次に勝手な点$b	\in	D$をとって，ここでも$f(b)=0$が成り立つことを見たい（☆☆）．
そのために，$a$と$b$を$D$内の連続曲線$z=z(t)	\:	(0	\leqq	t	\leqq	1)$で結ぶ．
十分小さい$t$については（$n\rightarrow	\infty$で）$z(t)$は$a$の近傍にあるから，今示した(☆)ように
$f(z(t))	=	0$が成り立っている．そこで$f(z(t))=0$が$0	\leqq	t	\leqq	t_0$で成り立つような
最大の$t_0$（☆☆☆）が$0	<	t_0	\leqq	1$の範囲にあるはずである．$t_0	=1$であることをがいえれば
$f(z(1))	=	f(b)	=0$となって目的（☆☆）を達する．仮に$t_0	<	1$であるとすれば，
$t_n	<	t_0,	t_n	\rightarrow	t_0	\:	(n	\rightarrow	\infty),	\:	z(t_n)	\ne	z(t_0)$と
なる列をとれば$f(z(t_n))=0$である（☆☆☆）から，上で示したことにより$f(z)=0$が$z(t_0)$の近傍で，
したがって$t_0$に十分近い$t	>	t_0$でも成り立つことになる．これは$t_0$のとり方に矛盾している．
\■

しばしば使うことだが，$z=a$で$f(z)=0$となるとき，$a$は$f(z)$の\textbf{零点}
であると言う．一致の定理より（対偶を取ると）

\系{2.39}
解析関数$f(z)$が恒等的に$0$でなければその零点は離散的である．
すなわち$f(a)=0$ならば，十分小さく$r>0$をとると$0	<	|z-a|	<	r$では
$f(z)	\ne	0$．
\□

一般に，領域$D_1$で定義された解析関数$f_1(z)$が，より広い領域$D_2$で定義された
解析関数$f_2(z)$に拡張できるとき，（詳しく言えば$D_1	\subset	D_2$かつ
$D_1$上$f_1(z)=f_2(z)$のとき）$f_2(z)$は$f_1(z)$の$D_2$への\textbf{解析接続}
であると言う．一致の定理は，解析接続があるとすればただ１つに決まってしまうことを示しているので，
これを\textbf{一意接続の原理}とも言う．

私たちは
\[
	\text{実関数}	
	\Longrightarrow	\text{テイラー展開}
	\Longrightarrow	\text{複素関数}
\]
というステップを踏んで，指数関数や３角関数を複素関数と見なした．
一致の定理によれば，そのような実際の手続きとは無関係に，解析関数として
の拡張は原理的に1つしかないことが保証されるわけである．

\例{2.40}
一致の定理に応用して，指数法則$e^{z+w}	=	e^z	e^w	\:	(z,w	\in	\C)$を
計算せずに証明してみよう．ただし$z,w$が実数のときこの式は既知とする．

まず$w$を実の定数とみる．このとき$e^{z+w},	e^ze^w$はともに領域$\C$上定義
された$z$の解析関数であり，$z$が実数の場合に一致している．
ゆえに一致の定理から両者はすべての$z	\in	\C$で等しい．
次に$z	\in	\C$を固定し，$w$を複素変数とみて同じ議論を繰り返せば，
結局任意の$z	,	w	\in	\C$で指数法則が正しいことがわかる．

指数法則の他の証明方法については演習問題を参照のこと．
\□

上の論法は応用が広い．一般に，ある領域$D$で定義された解析関数$f(z)$,
$g(z),	\cdots$の間の関係式(微分を含んでもより）
\begin{equation}
	F(f(z),g(z),	\cdots,	f'(z)	,g'(z),	\cdots)	=	0
	\label{eq2x35}
\end{equation}
を考えよう．ここに$F$は多項式である．(\ref{eq2x35})がある点の近傍
（あるいは実軸の区間）で成り立つならば，同じ関係は領域$D$全体で成り立つ．
この事実を指して，解析接続に関する\textbf{関数関係不変の原理}と言うことがある．
