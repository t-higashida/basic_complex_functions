\subsection{収束半径}
どんなべき級数(\ref{eq2x2})も$z=0$では1項しかないから自明に収束する．では
どのような$z	\ne	0$が収束するか？これに関して基本的なのは次の事実である．

\定理{2.1}
任意のべき級数(\ref{eq2x2})に対し，次のいずれかの1つが成り立つ．
\begin{list}{}{}
	\item	（ⅰ）すべての$z$に対し絶対収束．
	\item	（ⅱ）ある$\rho	>	0$があって，$|z|	<	\rho$において絶対収束，かつ$|z|	>	\rho$においては発散．
	\item	（ⅲ）$0$以外のすべての$z$に対して発散．
\end{list}
\□
すなわちそれぞれにべき級数は，自分に固有のある円板の内部$\{	z	\in	\C	|	|	z	|	<	\rho	\}$で
絶対収束する．これをべき級数(\ref{eq2x2})の\textbf{収束円}	(circle of convergence），
$\rho$をその\textbf{収束半径}(radius of convergence)という．
（ⅰ）は$\rho	=	\infty$（収束円は全平面$\C$)，（ⅲ）は$\rho	=	0$（収束円は空集合）である特別の場合
と考える．

（ⅱ）には等号が入っていないことに注意しよう．証明に入る前に例を挙げるが，
そこでわかるように$z$がちょうど収束円上$|z|	=	\rho$にあるときの収束・発散に関しては
いろいろな状況が起こり，一般には何も言えないのである．

\例{2.2}
次の級数は共通の収束半径$\rho	=	1$を持つ．
\begin{align}
		f_0(z)	&=	z	+	z^2	+	\cdots	+	z^n	+	\cdots	,		\label{eq2x3}\\
		f_1(z)	&=	\frac{z}{1}	+	\frac{z^2}{2}	+	\cdots	+	\frac{z^n}{n}	+	\cdots	,\label{eq2x4}	\\
		f_2(z)	&=	\frac{z}{1^2}	+	\frac{z^2}{2^2}	+	\cdots	+	\frac{z^n}{n^2}	+	\cdots	,	\label{eq2x5}
\end{align}
これらの級数の一般項は
\[
	|z^n|	\geqq	\frac{|z^n|}{n}	\geqq	\frac{|z^n|}{n^2}
\]
という関係にある．もし$|z|	>	1$ならば，$|z^n|/n^2	\rightarrow	\infty	\:	(n	\rightarrow	\infty)$であるから，
いずれの級数も発散する．また$|z|	<	1$ならば幾何級数(\ref{eq2x3})は絶対収束だから，他の２つも絶対収束．
以上から，収束半径は共通で$\rho=1$となる．

ちょうど$|z|=1$のときには微妙であって，
\begin{list}{}{}
	\item	（１）(\ref{eq2x3})は$|z|=1$のどの点でも発散，
	\item	（２）(\ref{eq2x4})は$|z|=1$で発散，$|z|=1$上のその他の点では収束，
	\item	（３）(\ref{eq2x5})は$|z|=1$上で絶対収束
\end{list}
となっている．実際（１）は$|z^n|$が$0$に収束しないことから，また（３）は
$\displaystyle	\sum_{n=1}^{\infty}	\frac{1}{n^2}$が収束することから明らかである．
（２）で$z=1$とした$\displaystyle	\sum_{n=1}^{\infty}	\frac{1}{n}$が発散することは
既知であるので，$z	\ne	1$としよう．$S_n	=	1	+	z	+	\cdots	+	z^n	\:	(S_0	=	1	)$とおけば，
\begin{align*}
	\sum_{n=1}^{N}	\frac{z^n}{n}
	=
	\sum_{n=1}^{N}	\frac{1}{n}	(S_n	-	S_{n-1})	
	&=
		\left(\frac{1}{1}S_1	-\frac{1}{1}	S_0\right)
	+	\left(\frac{1}{2}S_2	-\frac{1}{2}	S_1\right)
	+	\left(\frac{1}{3}S_3	-\frac{1}{3}	S_2\right)	\\
	&\quad	+	\cdots	
	+	\left(\frac{1}{n-1}S_{n-1}	-\frac{1}{n-1}	S_{n-2}\right)
	+	\left(\frac{1}{n}S_{n}	-\frac{1}{n}	S_{n-1}\right)
	+	\cdots	\\
	&\quad+	\left(\frac{1}{N-1}S_{N-1}	-\frac{1}{N-1}	S_{N-2}\right)
	+	\left(\frac{1}{N}S_{N}	-\frac{1}{N}	S_{N-1}\right)	\\
	&=
		\left(\frac{1}{1}	-\frac{1}{2}\right)S_1
	+	\left(\frac{1}{2}	-\frac{1}{3}\right)S_2
	+	\left(\frac{1}{3}	-\frac{1}{4}\right)S_3		\\
	&\quad	+	\cdots	
	+	\left(\frac{1}{n-1}	-\frac{1}{n}\right)S_{n-1}
	+	\left(\frac{1}{n}		-\frac{1}{n}\right)S_{n}
	+	\cdots	\\
	&\quad	+	
		\left(\frac{1}{N-1}	-\frac{1}{N}\right)S_{N-1}
	+	\frac{1}{N}S_N	-	S_0\\
	&=
	\sum_{n=1}^{N-1}	\left(\frac{1}{n}	-	\frac{1}{n+1}	\right)	S_n	+	\frac{1}{N}	S_N	-1	\\
	&=
	\sum_{n=1}^{N-1}	\frac{1}{n(n+1)}	\frac{1-z^{n+1}}{1-z}	+	\frac{1}{N}	\frac{1-z^{N+1}}{1-z}-1	
\end{align*}
と変形できる（\textbf{アーベル（Abel）の変形法）}.$|z^{n+1}|$は有界，また
$\sum_{n=1}^{\infty}	1/n	(n+1)$は収束するから（有界$\times$収束$=$収束）	，この式から(\ref{eq2x4})が$|z|=1,	z	\ne	1$で収束
することがわかる．
\□

さて定理2.1を証明しよう．

\補題{2.3} (Abel)　
べき級数(\ref{eq2x2})が$z=z_0$で収束すれば（絶対収束とは限らない），
$|z|<|z_0|$で絶対収束する．もし$z=z_0$で発散すれば$|z|>|z_0|$
でも発散する．

\証明
まず，次の注意から始めよう．ある正の実数$R>0$について
\begin{equation}
	|a_n	|	\leqq	\frac{M}{R^n}	\quad	(n	\geqq	0)	\quad	\text{となるように$M=M(R)$が選べる}
	\label{eq2x6}
\end{equation}
が成り立っていれば(\ref{eq2x2})は$|z|<R$において絶対収束する．実際
\[
	\sum_{n=0}^{\infty}	|a_n	z^n|	\leqq	\sum_{n=0}^{\infty}	M	\left(\frac{|z|}{R}	\right)^n
	=	\frac{M}{1-(|z|/R)}	<	\infty	
\]
となるからである．べき級数の収束を論じる際は，このように幾何級数と比較するのが常套手段である．

さて，(\ref{eq2x2})がもし$z=z_0$で収束すれば，$n	\rightarrow	\infty$で$a_n	z^n_0	\rightarrow	0$でなければ
ならない．特に，$|a_n	z_0^n	|	\leqq	M$がすべての$n$で成り立つように$M	>	0$を選ぶことができる．
$z_0	\ne	0$の場合だけを考えれば十分だが，このときは(\ref{eq2x6})が$R=|z_0|$として成立する．
ゆえに(\ref{eq2x2})は$|z|	<	|z_0|$で絶対収束している．

もし$z_0$では発散し，$|z|>|z_0|$なる$z$で収束しているとすれば（$z$と$z_0$の立場を入れ替えて考える），いま示したことに矛盾してしまう．
\■

そこで(\ref{eq2x2})が収束するような点$z_0$の絶対値の上限を
\begin{equation}
	\rho	=	\sup	\{	|z_0|	\:	|	\:	\text{(\ref{eq2x2})が$z_0$で収束}	\}
	\label{eq2x7}
\end{equation}
とおけば，$\rho	=	\infty	,	0$の場合もこめて確かに定理2.1が成り立つ．

\textbf{べき級数(\ref{eq2x2})}
\[
	a_0	+	a_1	z	+	a_2	z^2	+	\cdots	=	\sum_{n=0}^\infty	a_n	z^n
\]

\textbf{条件(\ref{eq2x6})}
\[
	|a_n	|	\leqq	\frac{M}{R^n}	\quad	(n	\geqq	0)	\quad	\text{となるように$M=M(R)$が選べる}
\]

上の証明からわかるように，
収束半径$\rho$はまた
\begin{align}
	0	<	R	<	\rho	\quad	&\text{なら(\ref{eq2x6})が成り立ち（$M(R)$が選べる），}	\notag	\\
	R	>	\rho			\quad	&\text{なら(\ref{eq2x6})が成り立たない（$M(R)$が選べない）}	\label{eq2x8}
\end{align}
という性質で特徴付けられる．係数の絶対値$|a_n|$が$n	\rightarrow	\infty$が速く増大するほど
収束半径が小さくなるわけである．

今後，収束半径が$0$でないべき級数を\textbf{収束べき級数}ということにする．
簡単だがつぎに事実に注意しておこう．

\命題{2.4}
収束べき級数$f(z)=\sum_{n=0}^{\infty}	a_n	z^n$の番号をずらして得られる級数
$\sum_{n=1}^{\infty}a_n	z^{n-1}	=	a_1	+	a_2z	+	a_3	z^2	+	\cdots$も
もとの級数と同じ収束半径を持つ．さらに$f(0)	=0,	f(z)	\not\equiv	0$ならば，
自然数$k$および$f(z)$と同じ収束半径をもつべき級数$\widetilde{f}(z)$があって
$f(z)	=	z^k	\widetilde{f}(z)	\ne	0$と書ける．

\証明
$z	\ne	0$ならば，$\sum_{n=1}^\infty	a_n	z^n$が収束することと$z^{-1}	\times	\sum_{n=1}^\infty	a_nz^n$
が収束することとは同値であるから，前半は(\ref{eq2x7})から明らかである．後半は，
$a_0	=	\cdots	=	a_{k-1}	=	0,	a_k	\ne	0$となる$k$をとって$\widetilde{f}(z)=	\sum_{n=0}^\infty	a_{n+k}z^n$
とおけばよい．
\■
