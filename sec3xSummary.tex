\subsubsection{＜まとめ＞}

\begin{list}{}{}
	\item	\textbf{3.1} 複素関数としての微分可能性は，実関数としての微分可能性に加えて
コーシー-リーマンの関係式を課すことを意味している．
	\item \textbf{3.2} 複素関数の積分は，関数が正則である限り積分路を変形しても値が
変わらない．
\end{list}



\subsubsection{演習問題}

\textbf{3.1}

次の関数$u(x,y)$を実部とする正則関数$f(z) \: (z = x+ iy)$を求めよ．

\begin{list}{}{}
	\item	(1) $u(x,y) = (x-y)(x^2 + 4xy+y^2)$
	\item (2) $\displaystyle u(x,y) = \frac{ \sin x }{\cosh y - \cos x}$
\end{list}

\textbf{3.2}

(1) $a,b >0$,$z = a \cos \theta + i b \sin \theta$とするとき
$\displaystyle \Img \left( \frac{1}{z} \frac{dz}{d \theta} \right)$を
$\theta$で表せ．

(2) 積分
\[
	I = \int_0^{\pi} \frac{d \theta}{a^2 \cos^2 \theta + b^2 \sin^2 \theta}
\]
を求めよ．

\textbf{3.3}

$a,b \: (a < b )$を実数，$f(z)$を多項式とし，次のようにおく．
\[
	F(z) = \int_a^b \frac{f(t)}{t-z} dt \quad ( z \in \C \setminus [a,b])
\]
\begin{list}{}{}
	\item	(1) 微分と積分の順序交換（本シリーズ『微分と積分 2』第1章）を利用して，
$F(z)$は$\C \setminus [a,b]$で正則であることを確かめよ．
	\item (2) 積分路の変形を利用して，$a < x < b$のとき
極限値$\displaystyle \lim_{\varepsilon\rightarrow0} F(x \pm i \varepsilon)$が存在し，さらに
\[
	\lim_{\varepsilon\rightarrow0} (F(x \pm i \varepsilon) - F(x - i \varepsilon)) = 2 \pi i f(x)
\]
が成り立つことを示せ．
\end{list}
